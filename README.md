# 📚 Fanfiki Garunu (Сборник фанфиков)

Добро пожаловать в репозиторий проекта **Fanfiki Garunu**! Это удобный веб-сайт для чтения фанфиков, объединяющий простоту HTML/CSS и серверную логику на Pascal.

## 🚀 О проекте

Проект представляет собой интерактивную читалку. Основная цель — сделать процесс чтения удобным, с приятным интерфейсом и быстрой навигацией.

**Основные возможности:**
*   📖 Удобный список историй.
*   🌓 Адаптивный дизайн (корректно отображается на телефонах).
*   ⚙️ Серверная часть на Pascal для управления данными.

Проект использует сочетание классического веба и необычного серверного решения:

## 📦 Как запустить локально

Чтобы проект работал корректно (со всеми стилями и серверными функциями), следуй инструкции:

1.  **Подготовка:**
    *   Установи [VS Code](https://code.visualstudio.com/) и расширение **Live Server**.
    *   Установи [PascalABC.NET](http://pascalabc.net/).
2.  **Запуск сервера:**
    *   Открой файл `server2 (1).pas` в PascalABC.NET и запусти его (нажми F9). Это поднимет локальный сервер для обработки данных.
3.  **Запуск интерфейса:**
    *   Открой папку проекта в VS Code.
    *   Нажми правой кнопкой на `index.html` -> **Open with Live Server**.
4.  **Готово!** Теперь можно переходить в браузер и читать истории.
